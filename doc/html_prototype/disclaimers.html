Drug Adverse Event Browser

Disclaimers

This application uses data from FDA through its openFDA initiative. OpenFDA provides APIs and is investigating raw download access to a number of high-value structured datasets, including adverse events, drug product labeling, and recall enforcement reports. The platform is currently in public beta.

OpenFDA is a beta research project and not for clinical use. While we make every effort to ensure that data is accurate, you should assume all results are unvalidated.

A report may list several drug products, as well as several patient reactions. No individual drug is connected to any individual reaction. When a report lists multiple drugs and multiple reactions, there is no way to conclude from the data therein that a given drug is responsible for a given reaction.

For more information on the openFDA project, visit open.fda.gov,

All trademarks (e.g., drug brand names) remain property of their respective holders and are used only as product data identifiers.
